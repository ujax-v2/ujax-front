## 🚀 변경 사항 (Changes)

1. **문제집 (Problem Box) UI 개편**
   - 문제집 리스트 카드의 폴더 아이콘과 컨테이너 배경 등 불필요한 시각 요소를 삭제.
   - 문제집 내부 접속 시 최상단 제목 하단에 문제집 소개 설명(`description`) 영역을 노출.
   - 문제집 생성(Create Box Form) 폼에 설명(`description`) 입력 필드 추가 및 상태(`ProblemBox` 타입) 바인딩.

2. **프로필 (My Page) UI & 데이터 렌더링 일체화**
   - 우측 상단 `개인정보 변경`, `회원탈퇴` 버튼 및 헤더의 `(My Page)` 영문 텍스트 등 불필요 요소 제거.
   - 좌측 "내 정보 상세"에 사용자의 로그인 수단(Provider) 명시용 캡슐 라벨(Badge) 추가 및 **참여 랩실(워크스페이스 수)**, **주력 언어**, **해결 문제 수** 등 애플리케이션에 적재된 상태값(recoil state) 기반의 파생 데이터 요소 추가 표기.
   - 우측 "활동 지표" 패널을 상하 확장(`h-[200px]`) 후 경험치(EXP) 진행 바를 삭제하고, **막대 그래프(`BarChart`)**를 활용한 “랭크 계급별(브론즈/실버/골드 등) 문제 해결 횟수” 시각화 차트 추가 배치.
   - 화면 좌상단에 이전 컨텍스트 혹은 즉시 최근 `Dashboard`로 복귀 가능한 "뒤로가기 화살표(Navigation Arrow)" 추가 및 여백 확보.

3. **타입스크립트(TypeScript) API 명세 동기화 및 에러 해결**
   - `@ujax/api-spec` 스키마 최신화 버전에 맞춰 `ProblemRegistration.tsx` 화면의 문제 조회 API 내부 필드명(`problemNumber`, `algorithmTags`) 갱신.
   - 각종 Auth 관련 렌더링, Local Storage 저장 시 누락되었던 `UserState` 인터페이스 내 `provider` 필드 엄격 정의 및 초깃값 바인딩을 통해 모든 TS 빌드 컴파일 오류 제거.

## 💡 의도 (Motivation & Context)
- **핵심에 집중하는 뷰 구성**: 구현 예정이 없는 가상 포인트, 경험치 요소보다 실질적인 문제 풀이 통계와 활동 내역(해결 횟수, 주력 언어 등)을 대시보드에 뿌려줌으로써 개발자 유저 입장에서 필요한 데이터를 한눈에 볼 수 있도록 의도했습니다.
- **불필요한 리소스 & 인지 부하 감소**: 문제집 카드의 화려한 폴더 박스, 사용되지 않던 옵션 등을 쳐내면서 한정된 컨테이너 내 시각적 밀도를 한 번 정리했습니다.
- 에러 해결 및 스키마 현행화를 통해 `dev` 서버 가동 시 나타나던 블로킹 이슈를 조치하여 향후 개발 편의성을 향상했습니다.
